# 📊 Salla Store Analysis Dashboard

## 📝 Project Overview
This project presents a comprehensive data analysis for **Salla E-commerce Store**. The dashboard provides actionable insights into sales performance and customer service efficiency (Call Center) to help drive data-driven decisions.

## 🛠️ Tools & Technologies
* **Power BI:** For dashboard design and data visualization.
* **Power Query:** For data cleaning and transformation (ETL).
* **DAX:** Used for creating advanced measures and KPIs.
* **Excel/SQL:** Data sources.

## 💡 Key KPIs & Insights
* **Sales Tracking:** Monitoring total revenue and order trends.
* **Call Center Performance:** Analyzing Handling Rates and Average Speed of Answer (ASA).
* **Interactive Filtering:** Ability to drill down by date, region, and agent.
